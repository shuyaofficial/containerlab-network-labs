物理・論理構成図

v1.0 ✅ 実機検証済 最終更新 2026-07-08 lab: nwzt-lan / mgmt 172.20.40.0/24
この図の役割：単一の core-sw（Cisco IOL）を中核に、802.1X 動的VLAN・inter-VLAN ACL・OpenZiti overlay・Suricata 監視を束ねた 実デプロイ済みラボの構成図です。緑＝実機で成立した部品／接続（B1〜B6 合格）、灰点線＝設計のみ・残課題（◐）。図中のノード名・IP・インターフェース名は nwzt.clab.yml / core_sw_merged.cfg と一致します。
緑＝実機で成立（✅／B1〜B6合格） 灰点線＝設計のみ・残課題（◐） 実線＝物理結線（Ethernet） 破線＝監視ミラー 点線＝ziti overlay ノードにホバー/フォーカスで名称・IP

物理視点＝結線トポロジ。IOL は 1スロット4ポートのため clab の eth4/5/6 が Et1/0 / Et1/1 / Et1/2 にマップされる（/iol/NETMAP 準拠）。監視SOC と ZTNA overlay は containerlab 管理外の docker 併走で、deploy-all.sh が一括起動する。
