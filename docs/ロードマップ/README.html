🚩 フェーズ1: ネットワーク構築の基礎（Cisco IOSベース） #

「企業の社内ネットワークを、要件定義 → 設計 → 構築 → 試験 まで一気通貫でやり切る」

まずは王道となる「Cisco IOSルーター・スイッチ」を使った構成で、インフラエンジニアとしての基礎を固めます。 L2/L3の冗長化、OSPF/BGP、基本的なセキュリティとQoSなどを含みます。 テーマ21〜22の統合演習に続き、テーマ23〜25でCollapsed Core、Routed Access、Multi-Campusを比較。 テーマ26〜27では動的ルーティングの深堀り（OSPF/EIGRP/BGP使い分け・再配信・NAT）と、QoS/ACL/route-mapの応用を社内LAN模擬環境で実践します。

👉 詳細カリキュラム・進め方はこちら（PHASE1_BASICS.md）

🚀 フェーズ2: モダン構成・拡張編（セキュリティ＆マルチベンダー） #

「実際の現場で使われるモダンなセキュリティ製品やクラウド管理を導入し、ゼロトラストへ拡張する」

フェーズ1のテーマ27までを土台とし、テーマ28以降で次世代ファイアウォール（FortiGate）、リモートアクセスVPN、NAC（802.1X）、ロードバランサ、NDR（可視化）などを順次実装・検証していく「シーズン2」です。

⚠️ フェーズ2はフェーズ1のネットワーク関連テーマを全て完了してから着手します。 サーバー系技術の学習はまだタイミングではなく、まずネットワーク層の基礎を盤石にする方針です。

👉 詳細テーマ一覧・拡張計画はこちら（PHASE2_MODERN_ENTERPRISE.md）
