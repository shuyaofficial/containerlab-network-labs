B. カタログ外で補完が必要な技術（重要） #

RFCカタログは IETF/L3中心 のため、社内LANの土台である以下は別途（IEEE規格・Cisco公式Configガイド）で学びます。 本ロードマップではこれを第2章で明示的に補完します。

技術 規格・出典 なぜ重要か
VLAN / 802.1Q IEEE 802.1Q 部署・用途でブロードキャストドメインを分割する社内LANの基本
STP / RSTP / MST IEEE 802.1D / 802.1w / 802.1s L2ループによるネットワーク全滅を防ぐ。最もトラブルが多い領域
EtherChannel IEEE 802.3ad (LACP) 帯域集約とリンク冗長
HSRP Cisco（RFC 2281 情報提供） デフォルトGWの冗長化（Cisco実務の定番）
VRRP RFC 5798 GW冗長の標準版（マルチベンダ）
ACL / NAT / PAT Cisco Config Guide 境界フィルタとアドレス変換
Syslog RFC 5424 運用ログ集約（ローカル未保持・必要時に取得）
