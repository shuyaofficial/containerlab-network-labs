第3章: ネットワークサービス（✅ 完了）

ルーティングとL2が整ったら、社内を「使える」状態にする土台サービスを載せる。 どれもカタログ「サービス・制御」の初級だが、実務での障害発生頻度は非常に高い定番。

🎯 章の狙い #

  • クライアントに自動で IP を配る（DHCP、サブネット跨ぎはリレー）
  • 名前解決（DNS）の階層委任とレコードの基本が分かる
  • 全機器の時刻を合わせる（NTP）＝ログ相関・証明書・認証の前提

前提 #

  • 第2章（VLAN/SVI）。DHCPリレーは Inter-VLAN 環境で真価を発揮する
