# 採点者専用フォルダ README

このフォルダ（`99_採点者専用_封印/`）は **ゲームマスター（採点AI）専用** です。プレイヤーが本フォルダ内のいずれかのファイルを開いた場合、該当するシナリオのプレイは無効となります。

## 収録ファイル

- `scenario01_模範切り分けフロー.txt`
- `scenario02_模範切り分けフロー.txt`
- `scenario03_模範切り分けフロー.txt`
- `scenario04_模範切り分けフロー.txt`
- `scenario05_模範切り分けフロー.txt`

各ファイルには対応シナリオの注入内容平文・症状全容・模範切り分けフロー・ヒント全文・採点アンカーが記載されている。GMはプレイ中の判定・ヒント提示・採点にのみこれらを参照し、内容をプレイヤーへ引用・示唆してはならない。
