# 切り戻し手順 — シナリオ04

## 実行手順

1. clab VM へログインする。

   ```
   ssh clab@orb
   ```

2. 本シナリオのフォルダへ移動する。

   ```
   cd シナリオ04_中級_回線切替失敗
   ```

3. 切り戻しスクリプトを実行する。

   ```
   sudo bash restore.sh
   ```

## 成功時の画面表示

進行メッセージが表示され、最後に「切り戻し完了。ベースライン健全性チェックを再実行してください。」と表示される（計画メンテ状態の解除も本スクリプトに含まれる）。

## 失敗時の画面表示

「切り戻し失敗。../02_ベースライン/切り戻し・完全リセット手順.md を参照してください。」と表示される。

## フォールバック（Level 2 / 3）

restore.sh で復旧できない場合は [../02_ベースライン/切り戻し・完全リセット手順.md](../02_ベースライン/切り戻し・完全リセット手順.md) のLevel2/3の手順に従うこと。

## 注意

スクリプトの中身を覗くとネタバレになるため開かないこと。
