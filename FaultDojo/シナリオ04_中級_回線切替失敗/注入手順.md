# 注入手順 — シナリオ04

## 実行前提

- ベースライン健全性チェックに合格していること（[../02_ベースライン/ベースライン構築手順.md](../02_ベースライン/ベースライン構築手順.md) 参照）。

## 実行手順

1. clab VM へログインする。

   ```
   ssh clab@orb
   ```

2. 本シナリオのフォルダへ移動する。

   ```
   cd シナリオ04_中級_回線切替失敗
   ```

3. 注入スクリプトを実行する。

   ```
   sudo bash inject.sh
   ```

## 成功時の画面表示

進行メッセージ（「計画メンテ状態を再現中...」を含む）が表示され、最後に「注入完了。ブリーフィングに従い切り分けを開始してください。」と表示される。

## 失敗時の画面表示

「注入失敗。切り戻し・完全リセット手順.md を参照してください。」と表示される。この場合は [../02_ベースライン/切り戻し・完全リセット手順.md](../02_ベースライン/切り戻し・完全リセット手順.md) を参照し、ラボの状態を確認・リセットすること。

## 注意

スクリプトの中身を覗くとネタバレになるため開かないこと。
