# テーマZERO｜構築ログ テンプレート

Phase を構築するたびに**このファイルを複製**し、`構築ログ_phaseN_YYYY-MM-DD.md` として作業を記録する。
記録の軸は「日時・作業・コマンド・結果・エラーと対処・学習ポイント」。秘密情報（初期 PW・鍵・トークン）は書かない。

| 対象 Phase | Phase N（観点） |
| 作業者 | |
