## ルーティングの要点
| 項目 | 内容 |
|---|---|
| Area 0 | Core1/2・FGT-Edge・各ディストリの上位リンク・全Loopback |
| Area 1（A棟） | VLAN10/20/901。ABR=Dist-A1/A2 が `10.10.0.0/16` に集約 |
| Area 2（B棟） | VLAN30/99。ABR=Dist-B1 が `10.20.0.0/16` に集約 |
| デフォルトルート | FGT-EdgeがArea 0へ広報 → 全社内機器がFGT経由でインターネットへ |
| 支社経路 | FGT-EdgeのVTI宛スタティックをOSPFへ再配布（10.2.40.0/24） |
| 利用者VLANのSVI | passive-interface（隣接はVLAN901とP2Pリンクのみ） |

## 通信ポリシー（要件2.3の論理配置）
| 制御 | 実装位置 |
|---|---|
| 社内→インターネット 許可＋SNAT | FGT-Edge |
| インターネット→社内 拒否 | FGT-Edge（ステートフル） |
| 営業⇔開発 制限 | Dist-A1/A2 のSVI着信ACL（**両方に適用**） |
| 支社→サーバVLANのみ許可 | FGT-EdgeのVPNポリシー（一元管理） |
| 支社→インターネット PAT | BR-Edge |
