# パラメータシート: QoS/ACL/route-map応用

本シートは「設定すべきパラメータ」のみを記載しています。具体的なCLIコマンドは受講者が構成します。

## 1. HQ Edge (hq-edge)
- **Interfaces**:
  - Loopback0: 10.27.255.254/32
  - Ethernet0/1 (to core): 10.27.1.6/30 (NAT inside)
  - Ethernet0/2 (to isp): 198.51.100.9/30 (NAT outside)
- **ACL (Security)**:
  - `OUTSIDE-IN`: Eth0/2のインバウンドに適用。
    - permit TCP any host 10.27.30.10 eq 80
    - permit TCP any host 10.27.30.10 eq 443
    - permit ICMP any any echo-reply (または established許可)
    - deny ip any any
- **QoS (LLQ)**:
  - `class-map match-any VOICE`: match dscp ef
  - `policy-map WAN-EDGE`: 
    - class VOICE: priority 2000 (kbps)
    - class class-default: fair-queue
  - Ethernet0/2 のアウトバウンドに `service-policy output WAN-EDGE` を適用。

## 2. HQ Distribution (hq-dist)
- **Interfaces**:
  - Loopback0: 10.27.255.2/32
  - Ethernet0/1 (to core): 10.27.1.2/30
  - Ethernet0/2 (to PC): 10.27.10.1/24
  - Ethernet0/3 (to VoIP): 10.27.50.1/24
- **PBR (Policy-Based Routing)**:
  - ACL: permit ip 10.27.50.0 0.0.0.255 any
  - `route-map PBR_VOICE permit 10`:
    - match ip address <ACL>
    - set ip next-hop 10.27.1.1
    - set dscp ef  (マーキングも同時に実施)
  - Ethernet0/3 のインバウンドに `ip policy route-map PBR_VOICE` を適用。

## 3. Branch Edge (br-edge)
- **BGP Route Filtering**:
  - `ip prefix-list BRANCH_NET`: permit 10.27.20.0/24
  - BGP process (AS 65001) 内で `neighbor 198.51.100.13 prefix-list BRANCH_NET out`。

## 4. HQ DMZ (hq-dmz)
- **ACL (Security)**:
  - `DMZ-OUT`: Eth0/2 (to srv) のインバウンドに適用。
  - DMZからTrust (10.27.10.0/24, 10.27.20.0/24) へのIP通信をdeny。
  - その他インターネット等への戻り通信はpermit。
