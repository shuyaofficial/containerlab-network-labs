- Linux 5台(wbitt/network-multitool): nms(.101, eth1=10.28.100.10/24) / cap(.102, IPなしpromisc) / pc-a(.103, 10.28.10.10) / pc-b(.104, 10.28.20.10) / srv-file(.105, 10.28.30.10)
- Zabbix 3台: zbx-db(postgres:16-alpine, .111) / zbx-srv(zabbix-server-pgsql 7.0.28 LTS, .112, eth1=10.28.100.20/24) / zbx-web(nginx-pgsql, .113, VMポート8280公開→Macから http://clab.orb.local:8280、初期 Admin/zabbix)
