# ネットワークラボ用 外部x86実行環境の選定

> 調査日: 2026-07-17。価格はすべて**調査時点の目安**であり、セール・為替・在庫で変動する。断定しない。
> 為替は **1ドル=150円と仮定**して換算（実勢レートで要再計算）。

## 1. Context

- 現環境は Apple Silicon M4 Mac (32GB) 上の OrbStack VM + containerlab（詳細は [環境説明.md](環境説明.md)）。
- KVMが使えないため、x86 VM型NOS（Cisco CSR1000v / C8000v / ASAv / NX-OS、Juniper vSRX、FortiOS 等 = **Tier3**）は
  QEMU TCG完全エミュレーションとなり**ブートに30〜90分**かかる（KVMネイティブなら2〜3分）。
- 目的: **KVMが使えるx86環境**を用意し、Tier3を実用速度にする。
- 運用形態: Macから ssh / docker context / VSCode Remote-SSH で操作。イメージ素材はGoogle Drive
  （`マイドライブ/Images/Dockerimege/`）に保有済みで、vrnetlabでビルドする。
- **判断軸は「Tier3を月に何時間使うか」**。Tier1/2（FRR・cEOS・IOL等のコンテナ型）は現Mac環境で足りており、
  x86環境はTier3専用の追加投資である。使用時間が少ないほど時間課金クラウドが有利、多いほど買い切りミニPCが有利。

## 2. 案1: 中古/新品 x86ミニPC

2026年7月時点で日本から入手しやすい候補。いずれもVT-x/AMD-V対応で**KVM可**。

| 機種（例） | CPU | コア/スレッド | RAM上限 | 実売価格帯(円・目安) | 消費電力 | 静音性 |
|---|---|---|---|---|---|---|
| Beelink Mini S12 Pro / S13 Pro | N100 / N150 | 4C/4T | 公称16GB(SODIMM×1) | 2.5〜3.0万(16GB/500GB) | TDP 6W・アイドル10W以下 | ほぼ無音 |
| 中古 ThinkCentre M720q/M920q Tiny・OptiPlex 3060〜7070 Micro | Core i5-8400T/8500T級 | 6C/6T | 公称32GB(SODIMM×2)・64GB動作報告あり | 1.5〜3.5万(リースアップ中古) | TDP 35W・アイドル10W前後 | 静か(小型ファン) |
| GMKtec NucBox M3 | Core i5-12450H | 8C/12T | DDR4 64GB | 定価5.998万・実売3.5〜5万 | アイドル10〜15W・負荷60W級 | 普通 |
| Beelink SER8 / Minisforum UM870 Slim | Ryzen 7 8845HS / 8745H | 8C/16T | DDR5 96GB(48GB×2) | 6〜9万(32GB/1TB) | アイドル10〜15W・負荷65W級 | 静音設計(SER8は32dB級) |

補足:
- **N100/N150級**: 最安だがRAM実質16GBのため、Tier3を同時1〜2ノード動かす入門用。NX-OS(単体で8〜12GB)は苦しい。
- **中古リースアップ品**（ThinkCentre Tiny / OptiPlex Micro）: 玉数が多く、Qualit・PC WRAP・イオシス・パセル等の
  中古専門店やヤフオク/メルカリで流通。本体1.5〜2.5万＋中古DDR4 SODIMM 32GB(2×16GB)5千円前後で**総額2〜3万**が組める。
  homelab定番のM720q/M920qはPCIeスロット増設例も多い。コスパ本命。
- **i5-12450H級**: 8コアでTier3を3〜5ノード。DDR4 64GBまで積めるため中規模ラボまで対応。
- **Ryzen 8845HS級**: 96GBまで積めるので、NX-OS複数＋C8000v複数の常設大規模ラボ向け。オーバースペック注意。

## 3. 案2: クラウド

### (a) 仮想EC2のネスト仮想化（2026年2月開始・本命）

**2026年2月、AWSがベアメタル以外の仮想EC2インスタンスでネスト仮想化(KVM)を正式サポート**した
（C8i/M8i/R8i/C7i/M7i/I7i系など。追加料金なし、起動時APIオプションで有効化）。
従来「クラウドでKVM=高額なmetal一択」だった前提が崩れており、ラボ用途ではこれが第一候補。

- 例: **m7i.2xlarge**（8vCPU/32GiB、東京 ap-northeast-1）: オンデマンド **$0.5208/時**、スポット **$0.193/時〜**
- 注意: 対応リージョン/世代は拡大中のため、東京リージョンでの対象インスタンス可否は利用前に要確認。
- GCPも従来からN2等でネスト仮想化可（東京 asia-northeast1 あり。n2-standard-8 は us-central1 で $0.39/時、東京は2〜3割高め）。
  AzureもDv3以降で対応。マルチクラウド比較は好みで。

### (b) ベアメタル系

| 選択肢 | 目安価格 | 備考 |
|---|---|---|
| EC2 metal (例: m5.metal 96vCPU/384GiB) | $4.6/時〜(us-east-1) | 東京はさらに高い。ラボには過剰・高額 |
| Vultr Bare Metal | $120/月〜(時間課金可) | 東京リージョンあり。常時確保なら選択肢 |
| Equinix Metal | — | **2026-06-30でサービス終了済み。選択肢から除外** |
| さくらの専用サーバPHY | 月額9千円〜＋初期費用9万円〜 | 国内・物理専有。2026-07-01に初期費用値上げ。ラボには初期費用が重い |
| Hetzner Server Auction (独/フィンランド) | €30〜39/月 | 欧州設置で日本からRTT 250ms級。ssh操作は可能だが体感は劣る。2026-06-15の値上げの対象外 |

### (c) 時間課金スポット運用の月額試算

EC2 東京 m7i.2xlarge（8vCPU/32GiB）＋EBS gp3 100GB（約$9.6/月・**停止中も課金**）で試算。1ドル=150円。

| シナリオ | 稼働時間/月 | オンデマンド | スポット |
|---|---|---|---|
| 週5時間 | 約22h | 約$11＋EBS ≒ **月3,100円前後** | 約$4＋EBS ≒ **月2,100円前後** |
| 週20時間 | 約87h | 約$45＋EBS ≒ **月8,200円前後** | 約$17＋EBS ≒ **月4,000円前後** |

- スポットは変動・中断リスクあり（ラボは再起動容易なので相性は悪くない）。
- イメージ再ビルドを避けるためEBSは残す前提。完全消去→都度ビルドならEBS分は削れるが手間増。
- 通信量・パブリックIP($0.005/時)等の小口従量が別途少額かかる。

## 4. 導入手順書（ミニPC想定）

前提: ミニPCにUbuntu Server 24.04 LTSをクリーンインストール（OpenSSH server有効、固定IPまたはDHCP予約）。

```bash
# 1) Mac側: 鍵ベースssh
ssh-keygen -t ed25519 -f ~/.ssh/clabx86   # 未作成なら
ssh-copy-id -i ~/.ssh/clabx86 user@<miniPC-IP>
# ~/.ssh/config に Host clabx86 を追記（HostName/User/IdentityFile）

# 2) ミニPC側: KVM確認（これが本題。可であることを最初に確認）
sudo apt update && sudo apt install -y cpu-checker && kvm-ok   # "KVM acceleration can be used" を確認
ls -l /dev/kvm

# 3) Docker（公式リポジトリ）と containerlab
curl -fsSL https://get.docker.com | sudo sh
sudo usermod -aG docker $USER && newgrp docker
bash -c "$(curl -sL https://get.containerlab.dev)"

# 4) vrnetlab（x86ではARMパッチ不要。上流をそのまま使う）
git clone https://github.com/hellt/vrnetlab.git ~/vrnetlab

# 5) イメージ素材の転送（Mac側から。Google DriveはMacにマウント済みのパスを使う）
rsync -avP "/Users/shuya/Library/CloudStorage/GoogleDrive-0vw329570d9444r@gmail.com/マイドライブ/Images/Dockerimege/" clabx86:~/images/
# プリビルドdockerイメージ(tar) → docker load -i xxx.tar
# qcow2素材 → ~/vrnetlab/<vendor>/ に置いて make（KVM有効なら数分/イメージ）

# 6) Macからの運用（clabはミニPC上で実行するのが基本）
ssh clabx86                       # tmux併用推奨
docker context create clabx86 --docker "host=ssh://clabx86"   # Macのdocker CLIから遠隔操作も可
# VSCode Remote-SSH でトポロジファイル編集。トポロジ(.clab.yml)はgit管理し、イメージとライセンスは載せない
```

### IOLライセンス(IOURC)の安全な移送

- **原則: 平文のIOURC/ライセンス文字列をgitリポジトリ・共有クラウドに置かない**（トポロジやentrypoint.shへの直書き注意。
  既存の`ensure_iourc`方式スクリプトをコミットする場合はライセンス部分を外部ファイル参照にする）。
- Mac→ミニPCは**scpで直接転送**（クラウドを経由しない）: `scp ~/secure/iourc clabx86:~/vrnetlab/... && ssh clabx86 chmod 600 ...`
- クラウド保管が必要な場合は**暗号化してから**置く: `age -p iourc > iourc.age`（またはgpg -c）。Google Drive上の既存
  `IOURC.zip`もパスワード無しなら暗号化版に置き換えを推奨。
- `.gitignore`に `iourc` / `*.lic` / `IOURC*` を追加してから作業を始める。

## 5. 判断基準（損益分岐）

電気代は目安単価31円/kWh（実勢は34円超の例もあり）。ミニPC 10〜25Wを24時間連続稼働しても**月約220〜560円**。

| 案 | 初期費用 | 月額(目安) | Tier3実用性 | 手間 |
|---|---|---|---|---|
| Mac完結（現状＋RAM 20GiB割当） | 0円 | 0円 | ×（ブート30〜90分） | 小 |
| ミニPC（中古Tiny＋32GB増設） | 2〜3.5万円 | 電気代 約200〜600円 | ◎（KVMネイティブ・常時待機） | 中（初期構築1日＋保守） |
| ミニPC（新品Ryzen 64GB級） | 6〜10万円 | 同上 | ◎（大規模ラボ可） | 中 |
| クラウド（EC2東京 nested・都度起動） | 0円 | 週5h: 2〜3千円 / 週20h: 4〜8千円 | ◎（ただし毎回起動/停止の段取り） | 小〜中 |

- **損益分岐**: 中古ミニPC総額3万円は、EC2オンデマンド週20h運用（月約8千円）なら**約4ヶ月**、週5h（月約3千円）なら
  **約10ヶ月**、スポット主体なら**1〜2年**で回収。
- 定性差: ミニPCは「思い立った瞬間に使える・データ常駐・従量課金の心理コストなし」。クラウドは「初期0円・
  故障/陳腐化リスクなし・使わない月はEBS代のみ」。

## 6. 結論の推奨

| Tier3使用頻度 | 推奨 |
|---|---|
| 月数時間以下 | **Mac完結を続行**（Tier1/2中心）。必要な回だけEC2スポットを単発起動 |
| 週5時間前後 | **まずEC2東京のnested対応インスタンス（スポット）で開始**。初期0円で試し、3ヶ月続いたら下段へ移行判断 |
| 週5〜20時間以上 | **中古ThinkCentre Tiny/OptiPlex Micro＋RAM32〜64GB（総額2〜4万円）が本命**。数ヶ月で回収でき、レイテンシゼロ・常設 |
| 常設大規模（NX-OS複数等） | Ryzen 8845HS級・RAM 64〜96GBの新品ミニPC |

### 確度が低い箇所（要再確認）

- ミニPC実売価格はセール変動が大きい（特にGMKtec/Beelink/MinisforumのAmazonクーポン）。購入直前に要確認。
- EC2ネスト仮想化の**東京リージョンでの対象インスタンス世代**は公式ドキュメントで最終確認のこと。
- スポット価格・為替（150円/ドル仮定）・電気料金単価は変動値。
- 中古Tinyの64GB RAM動作は機種/BIOS依存の報告ベース（公称は32GB）。
